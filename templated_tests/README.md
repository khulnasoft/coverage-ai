# Templated Tests
A series of tests used to validate the Cover Agent app under different languages and environments.

For more details see each subfolder for test projects.

## Systems / Low-Level
| Folder Name               | Language              | Description |
| ------------------------- | --------------------- | ----------- |
| `rust_cli`               | Rust                  | Command-line calculator with Cargo |
| `zig_cli`                 | Zig                   | CLI calculator with build.zig |
| `nim_cli`                 | Nim                   | Command-line calculator with Nimble |
| `c_kernel_module`          | C                     | Linux kernel module with proc interface |
| `cpp_game_engine`          | C++                   | Minimal game engine with CMake |
